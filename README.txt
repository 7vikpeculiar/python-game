The project is a terminal-based implementation of the game "Bomberman"

Objective : The player must defeat enemies and reach an exit to progress through levels. The player can place bombs, which explode in multiple directions after a certain amount of time, in order to destroy walls and kill enemies.

Controls:
w,a,s,d to move the bomberman up,left,down and right respectively.
b to place the bomb.
q to quit the game.

Requirements:
python3 and pip3 are required to run the game.
getch package is a must.
For installing getch ---> pip3 install getch

Steps to run the game:
To run the game use 
----> python3 board.py
